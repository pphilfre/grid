# GRID

Ultra-modern GEOINT/OSINT command center UI built with Next.js 15, Tailwind CSS, and shadcn-inspired components.

## Getting Started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.

## Tech Highlights

- Next.js 15 (App Router) + TypeScript
- Tailwind CSS with dark/light theming
- Component primitives aligned with shadcn/ui patterns
